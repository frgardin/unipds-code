# Orientaçao Objetos

## 1º pilar: abstracao

A abstração é o princípio que permite representar objetos do mundo real de forma simplificada, capturando apenas os aspectos essenciais e ignorando detalhes irrelevantes para o contexto do problema.

## 2º pilar: encapsulamento

O encapsulamento é o princípio que consiste em esconder os detalhes internos de implementação de uma classe, expondo apenas uma interface pública para interação e protegendo o estado interno do objeto.

## 3º pilar: heranca

A herança é um mecanismo que permite que uma classe (subclasse) herde propriedades e comportamentos de outra classe (superclasse), promovendo reutilização de código e a criação de hierarquias de classes.
## 4º pilar: polimorfismo

O polimorfismo é a capacidade de objetos de diferentes classes responderem de maneira diferente a uma mesma mensagem (método), permitindo que métodos com a mesma assinatura tenham implementações distintas nas subclasses.
